#### Integrations
##### Bitdefender GravityZone API
- Fixed **bd-push-settings-set** for partner/reseller API keys: Company ID is now passed as `subscribeToCompanies: [companyId]` (correct partner API structure) instead of a flat `companyId` parameter. When Company ID is not configured, `subscribeToCompanies` is omitted and events are received for all companies.
