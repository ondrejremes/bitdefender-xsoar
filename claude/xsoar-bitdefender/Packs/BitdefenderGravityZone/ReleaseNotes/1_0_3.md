#### Integrations
##### Bitdefender GravityZone API
- No integration changes.

#### Classifiers
##### Bitdefender GravityZone - Incoming Mapper
- **Malware events** (av, avc, hd, antiexploit, ransomware-mitigation, network-sandboxing): added mapping for Malware Type, Process Path, Parent Process Path, Exploit Technique, Source IP, Action Taken, MITRE ATT&CK.
- **Exchange Malware events**: added mapping for Email Sender, Email Recipients, Email Subject, Malware Type, Action Taken, Infected Object (file path). Computer Name now maps from `serverName` (exchange server).
- **EDR/XDR Incidents**: added mapping for Process Path, Parent Process Path, MITRE ATT&CK.
- **Network events** (fw, aph, network-monitor): added mapping for Application Path, Source IP, Local Port, Threat Name.
- **System events**: added `occurred` timestamp mapping.

#### Incident Fields
- Added **GravityZone Malware Type** — malware category (virus, trojan, adware, etc.).
- Added **GravityZone Process Path** — path of the detected/blocked process.
- Added **GravityZone Parent Process Path** — parent process path.
- Added **GravityZone Exploit Technique** — exploit technique used in antiexploit events.
- Added **GravityZone Application Path** — path of the application blocked by firewall.
- Added **GravityZone Source IP** — source IP of the attack or blocked connection.
- Added **GravityZone Local Port** — local port of the blocked connection.
- Added **GravityZone Email Sender** — sender address in exchange-malware events.
- Added **GravityZone Email Recipients** — recipient list in exchange-malware events.
- Added **GravityZone Email Subject** — email subject in exchange-malware events.
- Added **GravityZone MITRE ATT&CK** — MITRE ATT&CK technique ID.
