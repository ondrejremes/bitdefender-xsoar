#### Integrations
##### New: Bitdefender GravityZone API
- Initial release of the Bitdefender GravityZone API integration.
- Supports endpoint management (list, get details, isolate, de-isolate).
- Supports scan task creation and status tracking.
- Supports EDR/XDR incident note updates and status changes.
- Supports blocklist management (add, list, remove).
- Supports quarantine operations (list, remove, restore).
- Supports push event notification settings configuration and testing.
- Supports license information retrieval.
