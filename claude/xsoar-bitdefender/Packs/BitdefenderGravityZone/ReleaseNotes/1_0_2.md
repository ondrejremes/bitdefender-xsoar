#### Integrations
##### Bitdefender GravityZone API
- Added optional **Company ID** configuration parameter for partner/reseller API keys — automatically scopes push notification, quarantine, blocklist and license info calls to the specified managed company.
- Improved error messages: API errors now include the `data` field from the response when present.
- **bd-push-settings-get/set/test**: error `-32000` now includes a note that Push Notification Service must be enabled in GravityZone Control Center under My Company → Integrations.
- **bd-license-info**: error `-32601` now includes a note that `getLicenseInfo` is only available for partner/reseller API keys.
